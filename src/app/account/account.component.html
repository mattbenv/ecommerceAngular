Administrators shall only edit these properties if there is a request from a customer
or from a manufacturer or material provider.

Change Product Information, such as quantity, price, name, and variant information.
Product Information should only be changed in the case of external causes by admin

Edit Orders:

Change Order Information, such as user mistakes or cancelling an order.

Edit Categories:

Change Categories, or Remove a Category
